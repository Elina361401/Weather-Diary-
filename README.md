# Weather-Diary-
GUI-приложение "Weather Diary", позволяющее добавлять, просматривать и фильтровать записи о погоде, сохранять данные в JSON и использовать Git.
# Приложение "Weather Diary"

**Автор:** Тауканова Элина
**Назначение:** Локальный дневник погоды с возможностью фильтрации и сохранения данных.

---

## Описание программы
GUI-приложение на Python (Tkinter) для ведения записей о погодных условиях. Позволяет пользователям записывать дату, температуру, осадки и текстовые заметки. Все данные сохраняются в формате JSON.

---

## Как запустить приложение

### Требования
- Установленный интерпретатор **Python 3.8+**
- Стандартная библиотека Tkinter (поставляется вместе с Python)

### Запуск
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd weather-diary
   ```
2. Запустите скрипт:
   ```bash
   python weather_diary.py
   ```

---

## Примеры использования

1. **Добавление новой записи:**
   - Введите дату: `25.10.2025`
   - Введите температуру: `15`
   - Отметьте чекбокс "Осадки", если есть дождь/снег
   - Введите описание: `Ясно, легкий ветер`
   - Нажмите **"Добавить запись"**.

2. **Фильтрация записей:**
   - Для поиска записей за конкретную дату введите `25.10.2025` в поле даты фильтра и нажмите **"Применить"**.
   - Для поиска всех записей выше +10°C введите `10` в поле температуры фильтра и нажмите **"Применить"**.
   - Для возврата к полному списку нажмите кнопку **"Сбросить"**.
